# PatMeHonorific

## Installation

Installable using my custom repository (instructions here: https://github.com/anya-hichu/DalamudPluginRepo) or from compiled archives.

## Commands

 - `/patmehonorific info`
 - `/patmehonorific config`
 - `/patmehonorific enable`
 - `/patmehonorific disable`
